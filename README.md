# RheoMorph

A collection of all codes in conjunction with the project called RheoMorph at the ibvt. These include several topics including image analysis of microscopic images, the clustering of morphological data, the acquesition of rheological information and the modelling of the acquired data in relation to the production of secondary metabolites.

---
## File structure
---

### 🗂️ Modelling 

1. None-linear GPR models
2. A linear multivariate ridge regression model
3. FIM based sensitivity analysis

---

### 🗂️ Morphology

#### 🔬 MicroBioVision

Image Analysis for microscopic images of pellets. Calculates a plethora of geometric and size related features 

#### 📚 MorphCluster 

Clustering of pellet geometries

---

### 🗂️ Rheology

#### 🧾 Power Law Fit

Power Law Fit of Rheological data, calculating K and n, while also generating plots

---

### 🗂️ Tecan

Reads out the result files (.xlsx) of a Spark Multimode Microplate Reader. Can find all the data from fluorescence and absorpbance measurements with the same sample name.
